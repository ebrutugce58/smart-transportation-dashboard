# Smart Transportation Dashboard

This project was developed during the Anatolian Hackathon in collaboration with Yandex Türkiye.

## Overview
A web-based dashboard that predicts bus arrival times using machine learning and multiple datasets.

## Features
- Bus arrival time prediction
- Machine learning model using Random Forest Regressor
- Integration of route, stop, passenger flow, and weather data
- Web-based dashboard interface

## Technologies
- Python
- Flask
- Scikit-learn
- Pandas
- HTML
- CSS

## Machine Learning
The system uses a Random Forest Regressor to estimate bus arrival times based on historical and contextual data.
